# Environment Variables
Create a file called `.env` in **this** directory (no sub-folders) with the following lines:
```
TESSDATA_PREFIX = <full_path_to_tessdata>
DB_PATH = <path to db folder on your system>
GEMINI_API_KEY = <your Gemini API key>
SECRET_KEY =  <randomly generated secret key>
```
Tessdata can be cloned from https://github.com/tesseract-ocr/tessdata and the TESSDATA_PREFIX path must be the full path to the "tessdata" folder itself (e.g. D:/ethos/tessdata)
