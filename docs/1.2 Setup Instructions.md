The system is not currently deployed on the Internet, however the full system can be run locally, provided that the local device meets the server hardware requirements outlined in Section 3.1 of this report. The system consists of a Nodejs frontend and FastAPI backend, where both services must be started on the host machine. The frontend is accessed at `localhost:8080`.

For a technical and detailed guide on how to setup the system locally, follow the local deployment instructions in Section 3.2 of this report. Once these steps have been completed, the full system can be accessed from any modern web browser (Recommended: Chromium-based browsers, Safari or Firefox), at `http://localhost:8080`.

 Note: ensure that JavaScript is enabled in the browser (it is by default) before loading the page.
