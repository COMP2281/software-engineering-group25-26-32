Additional Features (From Requirements doc):

- Some sort of graphical visualisation of the relationships between theses, e.g. grouping them based on how similar they are to each other
- Improvements to the search model - make it use some of the full thesis text, e.g. to compensate for missing abstracts in the dataset.

Integration into EThOS
- EThOS currently offline - cannot do this until it comes back
- Would need to produce a database for EThOS containing the same field names as our durham e-theses db, and containing text content from 600,000 PDFS (!) - may require upgrading the database system (e.g. to Postgres instead of SQLite), and significantly increased storage requirements (~1TB+)
- Additional functionality for working with EThOS - need to add Welsh/Gaelic language support (since EThOS is a British collection, not just English), and devise a better method for extracting the text from the thesis documents since the Tesseract OCR cannot really handle handwritten scribbles.
