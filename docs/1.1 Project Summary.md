This report outlines the development of the EThOS Search Tool, including technical details of the implementation, installation and deployment instructions, and finally a test report evaluating the success of the project in meeting the client's requirements.

The primary aim of the tool is to build a system which allows researchers to search for information from the EThOS Collection - a collection of over 600,000 theses from British universities, dating back to 1700 - and make the information contained within the collection more accessible and useful to help propagate future research. The current system for searching the collection is basic and often returns results that are irrelevant to the search, meaning that a lot of time has to be spent manually reading through all the theses to filter out the relevant ones. This project aims to rectify this issue, with a focus on using AI to reduce the workload of human researchers when using EThOS.
