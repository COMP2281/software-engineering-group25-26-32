This report outlines the development of a semantic-based AI search tool which facilitates efficient and convenient research discovery and exploration of doctoral theses, including technical details of the implementation, installation and deployment instructions, and finally a test report evaluating the success of the project in meeting the client's requirements.

We are working with Professor Catherine Montgomery of Durham University to build a system which aims to allow researchers to search the EThOS Collection \- a collection of over 600,000 theses from British universities, dating back to 1700\. This will make the information contained within the collection more accessible, enabling this information to be used to propagate future research. The current system for searching the collection is basic and often returns results that are irrelevant to the search, meaning that a lot of time has to be spent manually reading through all the theses to filter out the relevant ones. This project aims to rectify this issue, with a focus on using AI to reduce the workload of human researchers when using EThOS.

As a result of a cyberattack on EThOS, our system was instead built around the Durham University E-Theses repository \- an alternative thesis repository suggested by our client to use during development while EThOS is offline. This thesis collection is significantly smaller than EThOS, with around 14000 theses, but system performance when scaled up to a larger dataset was considered throughout the development process. Once EThOS is restored, we plan to make the necessary changes needed to integrate the system into EThOS (see Section 4.2).
