**Hardware Requirements:**

Minimum Server Requirements:

| CPU | 2.8GHz clock speed, 4 cores (or better) |
| :---- | :---- |
| RAM | 8GB |
| Storage | 40GB for the provided Durham E-Theses dataset 500GB+ for EThOS |
| GPU | Nvidia GPU with CUDA support \- see [https://developer.nvidia.com/cuda/gpus](https://developer.nvidia.com/cuda/gpus)  |
| Operating System | Windows 10/11, MacOS, Linux |

Recommended Server Requirements:

| CPU |  |
| :---- | :---- |
| RAM | 16GB |
| Storage | 1TB |
| GPU | Nvidia GPU with CUDA support \- see [https://developer.nvidia.com/cuda/gpus](https://developer.nvidia.com/cuda/gpus) |
| Operating System | Windows 10/11 |

For detailed installation of the server software, see Section 3.2.

Client Requirements:

| Operating System | Windows, MacOS, Linux, Android, iOS |
| :---- | :---- |
| Software | Any Chromium-based web browser, Firefox or Safari Recommended: Google Chrome \- [https://www.google.com/intl/en\_uk/chrome/dr/download/](https://www.google.com/intl/en_uk/chrome/dr/download/) |
| Additional Requirements | Internet connection |

No additional software is required on the client-side. Ensure that your browser has JavaScript enabled, and navigate to the URL of the NodeJS web server. If running the server locally, this would be `localhost:8080`. Admin users should navigate to `localhost:8080/admin` to go to the Admin panel instead  
The API can be accessed by making requests directly to the URL for the API server. If running the server locally, this would be `localhost:8000`.
