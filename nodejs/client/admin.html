Admin Page





The DB Update button will automatically update a Durham Etheses database, or will extract text if the pdf_url field is used in the database.
The Build Index button will rebuild the index used for searching. This should be run after updating the DB.
Note: these operations can take a very long time to run.
Once complete, the updated file(s) will be available for download in the dropdown menu above. These can then be loaded into the live system via the Setup page.
