DB files must be SQLite databases with some of the following fields. CSV files must have some of the following fields. Names must be exact (case-sensitive). Other fields will be ignored.

Fields: title, author, abstract, award, faculty, department, keywords, date, url, pdf_url, pdf_text

Note: pdf_url must be a direct download link to a PDF. For other links (e.g. to the repository webpage containing information about a thesis) use the url field instead.
