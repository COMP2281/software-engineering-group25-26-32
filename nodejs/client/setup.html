DB or CSV File (.db/.csv):

IDs File (.npy):

DB files must be SQLite databases with the following fields. CSV files must have the following fields. Names must be exact (case-sensitive). Other fields will be ignored.
Required Fields: title, author, abstract, department, date
Optional Fields: url (link to thesis webpage), pdf_url (direct download link to PDF of thesis), keywords, award, faculty
The Index File and IDs File must match the uploaded dataset, and must have been generated using index.py.
Note: pdf_url must be a direct download link to a PDF. For other links (e.g. to the repository webpage containing information about a thesis) use the url field instead.
If you do not have the Index and IDs file for your database, you can generate them using the "Rebuild Index" button in the Admin panel, once you have uploaded your database above.
